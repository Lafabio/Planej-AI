# School Class Planning Tool

## Description
The School Class Planning Tool is designed to help educators plan and organize their lessons and class schedules efficiently. This tool offers a user-friendly interface to streamline class management and enhance the educational planning process.

## Features
- Easy-to-use interface for scheduling classes.
- Ability to manage multiple subjects and classes.
- Student management features to track attendance and grades.
- Reminder notifications for upcoming classes and assignments.
- Reports generation to analyze class performance over time.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Authentication:** JSON Web Tokens (JWT)

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Lafabio/Planej-AI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Planej-AI
   ```
3. Install dependencies for the frontend and backend:
   ```bash
   npm install
   cd client
   npm install
   ```
4. Start the application:
   - For the backend:
     ```bash
     npm start
     ```
   - For the frontend:
     ```bash
     cd client
     npm start
     ```

## Usage Guide
- After installing the application, open your browser and navigate to `http://localhost:3000` to access the tool.
- Sign up as a new user or log in if you already have an account.
- Follow the on-screen instructions to start scheduling your classes and managing your students effectively.
